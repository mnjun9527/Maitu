## 一、 安抚顾客
### 当顾客明显情绪过激如何应对？

::: warning 注意
当顾客明显情绪失控、抱怨、或吐糟时,安抚是第一步<br>
但是必须注意：`安抚不能代替解决方案`，我们不仅要安抚顾客,更要引导顾客进行`核实`以及`提供售后方案`！
:::

::: danger 异常行为
1. 过渡安抚,丢弃了`售后的核心目标`,`未引导核实`,以及没有提出任何`解决方案`.
2. 被顾客的情绪牵动,导致沟通停滞,`售后流程停滞不前`.
:::


::: details 安抚短语参考
```c#
1.亲的心情我们理解呢，您先不要太过于着急 希望您给点时间，我们为您加急处理中呢
2.亲，没有做到让您满意，十分的抱歉，不管问题如果解决，都不希望影响到您的心情
3.非常理解您的心情，请您先别生气，我这边会尽力帮助您的。
4.亲，这边我在帮您催促加急一下，争取不浪费您过多的时间和精力，也麻烦您耐心等待一下呢~
5.亲，在现实中客服也是一名消费者，所以亲，您的心情我们可以理解，还是希望亲亲能再给我们一点时间，我会尽快给您处理的
6.亲爱哒，我们已经给您加急催促了哦，都会给您优先处理的~
7.给您带来这么多的麻烦时针是非常抱歉，如果我是您的话，我也会很生气的，请您先冷静一下。
8.亲亲，明白您非常着急，我们特别为您加急催促，辛苦您再等下呢~麻烦了~
```
:::



## 二、 核实产品问题
### 如何确认问题？
当顾客反馈产品有问题时,我们需要引导顾客提供问题的具体证明,如`拍照`或`拍视频`的方式<br>
这一步可以快速帮助我们核实问题的真实情况.

::: warning 注意
- 判断商品是否存在`质量问题`
- 如果存在质量问题,及时提供`解决方案`
- 如果没有质量问题,需解释并`引导顾客正确使用产品`,如果顾客`难以接受`,可以适当进行`优惠/补偿`.
:::

::: details 核实短语参考
```c#
1.麻烦您拍摄一下商品存在的问题,这边会尽快为您解决,请你你放心!
2.亲，您将您的问题拍摄给我们，我们才能更好的为您服务，帮您解决问题呢。
```
:::


## 三、 提供售后方案
### 售后方案概述
当核实顾客的产品存在质量问题后，应及时提供`售后方案`<br>
**售后方案是与顾客协商解决问题的主要渠道!**   --->快速进入这个环节，是解决售后问题的关键

#### 1. 售后方案包含哪些？
| 售后方案           |      描述      |
| -------------- | :-----------: |
| 补偿方案       |通常情况下是解决问题`最快的方式`,在补偿额度内,`优先级最高`。|
| 补发方案       |用于配件缺少或损坏,优先补发，也可以配合补偿方案一起解决。|
| 换货方案       |当前两种方案无法解决，或者顾客不接受的情况下,换货可以`解决绝大部分的`问题.|
| 退货方案      |退货方案优先级最低,需顾客主动突出,且有注意事项.| 


## 四、 售后方案详解
### 1. 补偿方案！
#### 适用场景

* `污渍`,`瑕疵`,`轻微磨损`等`不影响使用的场景`, 以及`顾客对商品不满`等, 都可以采取补偿方案 <br>

#### 补偿额度
* 商品在500元以下,补偿额度范围为`实收金额20% 以内`(包含晒图在内)
* 具体金额见对应小组,【飞书群】`售前售后让利表`

#### 举例
* 售后场景1：
    - 产品：电热毯
    - 实付金额:218元
    - 顾客反馈问题:电热毯有污渍  -->**不影响使用**,协商补偿解决
    - 最大补偿额度:218*20%=43.6元 (含晒图在内)

* 售后场景2：
   - 产品：电热毯
   - 实付金额:218元
   - 顾客反馈问题:不通电  -->**影响使用**,`补偿方案不适用`！

#### 补偿话术参考
::: details `补偿短语参考`
```c#
1.抱歉亲,没能让您收到满意的产品,您试一下机器能不能正常使用,能正常使用我们补偿您xx元可以吗？
2.亲亲，非常抱歉，您看这边给您申请一些补偿，您看是否可以接受呢？
```
:::
---

### 2. 补发方案
#### 适用场景
* 适用于配件漏发、破损、等其他通过补发配可以解决的场景. <br>
* 在顾客有异常情绪的场时,通常可以`配合补偿方案一起使用`.

#### 举例
* 售后场景1：
    - 产品：热水龙头
    - 实付金额:199元
    - 顾客反馈问题:购买漏保款、没有收到漏保  --> **补发匹配可以解决**,协商补发配件
    - 有情绪的顾客可以,补偿+补发,不接受的情况下，引导换货方案

* 售后场景2：
    - 产品：热水龙头
    - 实付金额:199元
    - 顾客反馈问题:产品不通电  --> **补发配件无法解决**,协商其他方案

#### 补发方案术语参考
::: details `补发短语参考`
```c#
1.抱歉呢亲,这边给您补发对应的配件,所有补寄的费用都是我们承担,请您放心
2.十分抱歉,给您带来不好的体验，这边给您补发配件再给您申请补偿10元,您看可以吗？
```
:::

---
### 3. 换货方案
#### 适用场景
* 适用于`质保期内`当商品存在`质量问题`，`并且影响使用`的时候，优先采取此方案<br>


#### 举例
* 售后场景1：
    - 产品：热水袋
    - 实付金额:39元
    - 顾客反馈问题:热水袋破损、漏水  -->`影响使用` --> 优先采取换货方案

* 售后场景2：
    - 产品：热水袋
    - 实付金额:39元
    - 顾客反馈问题:热水袋不喜欢,没有达到预期 -->非质量问题，换货无法解决

#### 换货术语参考
::: details `换货短语参考`
```c#
亲，出现这样的问题真的非常抱歉，您放心我们一定会为您妥善处理。您看我们给您换货可以的吗，来回运费我们承担的，请您放心！
```
:::


---
### 4. 退货方案

#### 适用场景
* 需要满足退货条件 -->7天无理由 (如有特殊约定，以约定为准)
* 此方案由`顾客主动提出`，客服在非必要情况下`不主动提及退款`。<br>

#### 退货流程事件
* `劝留事件`：当顾客提出退货时,立即进行劝留 (即使顾客百分百不会留下,也要执行此动作)
* `原因引导事件：`当顾客提出退货时,主动引导原因，并且帮助顾客发起退货申请,选择七天无理由、其他等`不影响店铺评分的选项`

#### 举例
* 售后场景1：
    - 产品：热水袋
    - 实付金额:39元
    - 购买时间:7天内
    - 顾客反馈问题:`热水袋不通电`,`顾客要求退货`！

::: warning 此时触发 `质量问题劝留` 及`原因引导` 事件:
::: code-group

```c#[质量问题劝留]
1.亲亲，实在抱歉给您添麻烦了,您看能不能给咱们一次机会,我们帮您安排换一下可以吗亲,运费我们承担呢。
2.亲，非常抱歉给您添麻烦了,但是请您放心,我们店铺都是有保障的,我们承担来回运费给您换货,另外为了表示歉意,我们再给您补偿    元,您看下可以吗？
```

```c#[原因引导]
亲爱哒为能更快为您处理退货，请您填写退货原因为“七天无理由退换货/其他”，问题描述填写退货或者换货，仓库会加快您的审核，感谢您的支持 
```
:::

* 售后场景2：
    - 产品：电热毯
    - 实付金额:199元
    - 购买时间:7天内
    - 顾客反馈问题:电热毯不喜欢,想要退货

::: warning 此时触发 `非质量问题劝留` 及`原因引导` 事件:
::: code-group
```c#[非质量问题劝留]
1.亲，非常抱歉没有让您满意，这边给您申请补偿  元，亏本给您处理了呢亲，您看是否可以留下呢
```
```c#[原因引导]
亲爱哒为能更快为您处理退货，请您填写退货原因为“七天无理由退换货/其他”，问题描述填写退货或者换货，仓库会加快您的审核，感谢您的支持 
```
:::

::: danger 注意事项
注意,当顾客提出退货时,应先核对`退货时效`！,做好`劝留`话术推送以后,及时发起`退换货服务单`<br>
选择**非质量问题选项**,协助顾客完成退货申请！
:::


## 四、 备注

**1.** 备注的重要性：
* 所有的咨询顾客,除售前咨询以外,所有的问题都应有备注记载,无备注等于0产能。

**2.** 备注格式：
* 问题描述+进度

**3.** 不备注可能造的损害：
* 顾客反复描述 -->导致顾客情绪过激
* 客服重复沟通 -->交接沟通困难,增加工作量
* 资金损失,或处理错误

**4.** 备注范例:

```c#
1.商品瑕疵，协商补偿10元未发,顾客未回
2.划痕，补偿10元已发
3.反馈漏水，待顾客发视频
4.补发漏保，已经登记
5.非质量问题，退货运费自理
6.顾客要求补偿200，已拒绝，协商补偿50元
7.催退款，已经催促后台处理
8.顾客催快递、催派件，已经通知对应快递群
....
```
