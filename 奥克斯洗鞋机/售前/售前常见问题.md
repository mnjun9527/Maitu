# 洗鞋机前常见问题

## 一、需要安装吗？ 包安装吗？ 上门安装吗？

```Python
亲，咱们的洗鞋机不需要专业安装的哦，洗鞋机有两种款式：
半自动的：收到货后放到您选定的位置，连接上电源就可以了
全自动的：收到货后把进水管简单的连接到水龙头上，在连接上电源就可以了哦，非常简单的呢
```

## 二、毛刷不拆可以洗袜子吗？

```Python
亲、洗鞋机专门是用来洗鞋的，不拆毛刷，也是可以洗的
相对来说效果没有洗鞋子这么好,所以建议是拆掉毛刷清洗呢
```

## 三、洗鞋机要不解鞋带？

```Python
亲爱的，鞋带不解的话，可能会有缠绕毛刷的风险哦，平时不解也不会有很大影响呢。
```
